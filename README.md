## 4. 기능 및 역할

풋볼을 좋아하는 사람들의 가장 큰 문제는 사람을 구하는것부터 시작합니다. 최소 10명의 인원이 모여야 게임을 진행할 수 있기 때문에 사람을 구하기 어려울 수 있습니다.
모두의 풋볼은 이런 사람들을 위해 웹에서 사람들을 모아 게임을 진행할 수 있는 편의를 제공합니다.
